visible

  • get visible(): boolean

  • Inherited from Marker.visible

    • Defined in viewer/scene/marker/Marker.d.ts:135

    Indicates if this Marker is currently visible.

    This is read-only and is automatically calculated.

    The Marker is invisible whenever:

    • Marker.canvasPos is currently outside the canvas,
    • Marker.entity is set to an Entity that has Entity.visible false, or
    • or Marker.occludable is true and the Marker is currently occluded by an Entity in the 3D view.

    The Marker fires a "visible" event whenever this property changes.

    Returns boolean
